Overview (HTML)

HTML

  • What is HTML?
    • HTML stands for Hypertext Markup Language.
    • Document-layout and hyperlink-specification language used to create web pages.
    • Unlike a scripting or programming language that uses scripts to perform functions, a markup language uses tags to identify content.

Basic HTML Document Structure

  • The HTML Tag
    • Defines the root of an HTML document.
    • Although various version has been released over the years, HTML basics remain the same.
    • These tags, like the bread in a sandwich, should surround everything that is written using the HTML language
  • The HEAD Tag
    • Defines information about the document, including metadata and scripts.
    • The head of an HTML file contains all of the non-visual elements that help make the page work.
  • The BODY Tag

    • Usually follows the HEAD tag.
    • All visual-structural elements are contained within the tag pair, such as headings, paragraphs, lists, quotes, images, and links
